## Keywords With ZERO Search Volume — Do Not Target

❌ Specific event types: conference photographer nyc, gala photographer nyc, networking event photographer, trade show photographer
❌ Industry-specific: startup photographer, nonprofit photographer, tech event photographer
❌ Quality modifiers: best event photographer nyc, top videographer, affordable
❌ Package terms: photo and video package nyc, full event coverage
❌ Neighborhood targeting: manhattan photographer, midtown, downtown

---

## Secret Weapon

**"Corporate video production NYC"** — 500 searches/month, competition score 3/100.
Almost no one is targeting this. Always position as a "production company" not just a "videographer."

---

## Page Strategy

**Do this:**
- Build 2–3 excellent service pages (not 20 thin ones)
- ONE comprehensive event coverage page mentioning all event types in body copy
- Include pricing sections on every service page (people search for pricing)
- Use "transparent pricing" and "fixed packages" language throughout

**Don't do this:**
- Separate pages for conference, gala, networking (zero search volume)
- Neighborhood pages (Midtown, Downtown — zero volume)
- Industry pages (tech, nonprofit — zero volume)

---

## Pricing Language (Targets "corporate event photography pricing")

Use phrases like:
- "Starting at $X for [service]"
- "Transparent pricing with no hidden fees"
- "Fixed packages, no surprise charges"
- "All pricing includes [deliverables]"

---

## Meta Tag Rules

- Title: 50–60 characters, primary keyword included
- Description: 150–160 characters, action-oriented
- One H1 per page, primary keyword in H1
- Primary keyword in first 100 words of body copy
- 2–4 keyword mentions per 500 words (natural, not stuffed)

---

## Internal Linking Rules

- Blog posts → link to service pages (high priority)
- Service pages → link to portfolio and contact
- Portfolio → link back to relevant service pages
- Every page should have 3–5 internal links

---

## Technical SEO Checklist (Each New Page)

- [ ] Unique title tag (50–60 chars, keyword included)
- [ ] Unique meta description (150–160 chars)
- [ ] One H1 with primary keyword
- [ ] Canonical URL tag
- [ ] Open Graph tags (og:title, og:description, og:url, og:image)
- [ ] Article or LocalBusiness schema where appropriate
- [ ] Breadcrumb schema
- [ ] Added to `sitemap.xml`
- [ ] Linked from at least one other page
- [ ] Image alt text: descriptive + keyword where natural
