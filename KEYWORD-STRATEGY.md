### üèÜ The Secret Weapon: "Corporate Video Production NYC"

## Content Writing Guidelines

### Keyword Density (Natural Integration)

**Target:** 2-4 mentions per 500 words

**Good example:**
"We provide event photography throughout NYC, covering corporate events, conferences, and company gatherings."

**Bad example:**
"Event photographer NYC event photography NYC provides the best event photographer services in NYC for NYC events."

### Pricing Language (Target: "corporate event photography pricing")

Since people search for pricing, use phrases like:
- "Starting at $X for [service]"
- "Transparent pricing with no hidden fees"
- "Get a fixed quote in 24 hours"
- "All pricing includes [deliverables]"
- "No hourly rates or surprise charges"

### Event Types (Mention ALL in body copy)

While specific types have zero search volume, mention them for topic authority:
- Corporate conferences
- Annual galas
- Fundraising events
- Networking events
- Product launches
- Company parties
- Awards ceremonies
- Trade shows
- Panel discussions
- Team building events

**Why?** Google's algorithm understands topic clusters. A comprehensive page about "event photography NYC" that covers various event types will rank better than a thin page.

- [ ] Primary keyword in first 100 words

**Images:**
- [ ] Descriptive alt text (include location when relevant)
- [ ] File names include keywords (event-photography-nyc-conference.jpg)
- [ ] Optimized file size (WebP format preferred)

**URL Structure:**
- [ ] Clean, keyword-rich URLs (/event-coverage, not /page1)
- [ ] Hyphens (not underscores)
- [ ] Lowercase only

**Essential:**
- [ ] XML sitemap created and submitted to Google Search Console
- [ ] Robots.txt configured

- [ ] Mobile-responsive design
- [ ] Fast loading (Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1)
